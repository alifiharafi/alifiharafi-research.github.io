Perancangan Model Data Relasional pada Situs Web Perguruan Tinggi dengan Optimisasi Mesin Pencari

Relational Data Modeling on Higher Education Website with Search Engine Optimization

Penelitian ini bertujuan untuk membangun rancangan model data relasional pada situs web perguruan tinggi disertai dengan dukungan optimisasi mesin pencari. Luaran yang dihasilkan yaitu Model Data Konseptual dan Model Data Logis yang dapat dioperasikan pada RelaX. Lebih lanjut dapat mengakses direktori higher-edu-web.
